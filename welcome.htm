VIDOOMY REPORTS SERVER (LEGACY)
  • Endpoint to advertisers reports: http://local.reports.vidoomy.com:8080/reports_/adv/repo.php
  • Endpoint to publishers reports: http://local.reports.vidoomy.com:8080/reports/repo.php
PREREQUISITES
  • docker
  • docker-compose
  • run: sudo apt install build-essential
INSTALLATION INSTRUCTIONS
  1. git clone https://github.com/vidoomymedia/report-server.git
  2. cd report-server
  3. make init
  4. make php-shell
  5. composer install
  6. ask for DB dumps

    tables required for DB advertisers: (vidoomy_adv)

    • agency
    • campaign
    • reports
    • country
    • users
    • advertiser
    • ssp
    • dsp

    tables required for DB publishers: (vidoomy_login)

    • reports202106
    • reports_resume202106
    • supplytag
